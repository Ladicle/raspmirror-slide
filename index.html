マジックミラーとは?

    ガラス等の透明素材の上に金属皮膜を薄くして
    入射角の半分のみ透過したもの

RaspMirrorの仕組み

  • 液晶画面 On ⇢ 液晶側が明るく映像は透過
  • 液晶画面 Off ⇢ 外が明るく光は反射し鏡に見える

RaspMirrorの成分

  • ポリカハーフミラー
  • RaspberryPi Model2 typeB
  • 10インチ液晶
  • 黒画用紙
  • 両面テープ

RaspMirrorができるまで

  1. 液晶画面を組み立てる
  2. 黒画用紙に両面テープを貼る
  3. 液晶画面をマジックミラーの任意の場所に置く
  4. 液晶画面を覆い被すように黒画用紙をマジックミラーに貼る
  5. RaspberryPiの設定
  6. ミラー表示用のアプリを開発する

RaspberryPiの

事前設定

$ cat /etc/xdg/lxsession/LXDE-pi/autostart@lxpanel --profile LXDE-pi@pcmanfm --desktop --profile LXDE-pi@xscreensaver -no-splash@xset s off@xset -dpms@xset s noblank

#2 全画面用ブラウザの "INSTALL"

sudo apt-get install unclutter

#3 カーソル非表示用PKGの "INSTALL"

sudo apt-get install chromium-browser
